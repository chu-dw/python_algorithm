## 문제
한 마을에 모험가가 N명 </br>
모험가 길드에서 N명의 모험가를 대상으로 공포도 측정 </br>
공포도 높은 모험가는 위험사항 대처 능력 떨어짐 </br>
따라서 공포도가 x인 모험가는 반드시x명 이상으로 구성한 모험가 그룹에 참여해야함 </br>
이때 최대 몇개의 모험가 그룹을 만들수 있는가 </br>

## 입력
첫째 줄에 모험가의 수 N 주어짐 (1<N<100,000) </br>
둘쨰 줄에 각 모험가의 공포도의 값을 N이하의 자연수로 주어짐 각 자연수는 공백으로 구분 </br>

## 출력
여행을 떠날 수 있는 그룹 수의 최댓값을 출력 </br>

## 풀이1
```python
N = int(input())

scaredList = list(map(int,input().split()))
print(N,scaredList)

scaredList.sort()
print(scaredList)

count = 0

while scaredList:
  scared = scaredList.pop()

  for i in range(scared-1):
    scaredList.pop()
  count += 1
  print(scaredList)


print('c',count)
```
- 문제점1 : 모험가를 제외해도 된다 (문제에는 안주어졌는데.. 당연한건가?)  </br>
일단 모든 모험가를 넣는다 생각하고 최대를 만들려다 보니 공포도 큰 모험가 부터 처리를 했다.
- 문제점2 : 반복문에있는 pop  </br>
남은 scareList 길이 확인없이 꺼낸 만큼 pop을 해서 index에러가 나는 경우가 있다.

## 풀이2
```python
N = int(input())

scaredList = list(map(int,input().split()))
scaredList.sort()

count = 0
groupCnt = 0

for s in scaredList:
  count+=1
  if s <= count:
    groupCnt += 1
    count = 0

print(groupCnt)
```

간단하게풀어진다.
최대값을 구해야하기 때문에 공포도가 작은애들 부터 그룹을 만드는것을 코드로 보면 </br>
오름차순 정렬 해 놓은뒤 처음 부터 방문을 한다. 이때 count를 하나씩 올려주는데 </br>
이 count가 방문한 공포도 보다 같거나 더 크면 그룹을 생성하고 count를 초기화 한다. </br>
지나온 사람의 수가 공포도보다 높거나 같아진 것이기 때문에 지금까지를 하나의 그룹으로 묶는 것이다. </br>
