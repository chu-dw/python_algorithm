# 볼링공 고르기

## 문제
두 사람이 볼링을 친다. 두 사람은 서로 무게가 다른 볼링공을 고르려한다.
볼링공은 총 N개가 있으며, 볼링공마다 무게가 적혀있고, 공의 번호는 1번부터 순서대로 부여된다.
볼리공 무게는 1부터 M까지 자연수 형태이다.
N이 5이고, M이 3이며 각각의 무게가 차례대로 1, 3, 2, 3, 2 일때
각 공의 번호가 차례대로 1번부터 5번까지 부여된다.
이때 두사람이 구할 수 있는 볼링공 번호의 조합을 구하면
(1,2) (1,3) (1,4) (1,5) (2,3) (2,5) (3,4) (4,5) 이다.
결과적으로 두사람이 공을 고으는 경우의 수는 8가지이다.
N개의 공의 무게가 주어질때 두 사람이 공을 고르는 경의 수를 구하라.

## 입력조건
첫째 줄에 볼링공의 개수N, 공의 최대 무게 M이 공백으로 구분되어 각각 자연수의 형태로 주어진다.
둘때 줄에 각 볼링공의 무게 K가 공백으로 구분되어 순서대로 자연수 형태로 주어진다.

## 출력조건
첫째 줄에 두 사람이 볼리공을 고르는 경우의 수를 출력한다.

## 풀이
일단 그리디기 때문에 단순하게 생각하여 이중 반복으로 풀어봤다.

```python
N, M = map(int,input().split())
ball = list(map(int,input().split()))

idx = 0

print(ball)
count = 0

for i in range(N):
  for j in range(i+1,N):
    print(i, ball[i])
    print(j, ball[j])

    if ball[i] == ball[j]:
      print("같음")
    else:
      count+=1

print(count)
```

필요한 아이디어는 무게별 개수를 구하는 것이다.
