# 만들 수 없는 금액
## 문제
N개의 동전 가지고있음, N개의 동전을 가지고 만들 수 없는 양의 정수 금액중 최값을 구하라 </br>
예를들어 N=5이고, 각 동전이 각각 3원 2원 1원 1월 9원이라 가정한다. </br>
이떄 만들수 없는 양의 정수 금액중 최소는 8원이다. </br>

## 입력
첫째 줄에는 동전의 개수를 나타내는 양의 정수N (1 < N < 1,000) </br>
둘째 줄에는 각 동전의 화폐 잔위를 나타내는 N개의 자연수가 주어지며, 각 자연수는 공백으로 구분 </br>
이때 각  화폐의 단위는 1,000,000 이하의 자연수 </br>

## 출력
첫째줄에 주어진 동전들로 만들 수 없는 양의 정수 금액중 최솟값을 구하라

## 풀이
접근방법이 떠오르징않아 약간의 힌트를 받았다. </br>
- 지금까지 만들 수 있는 금액의 범위”를 관리한다 </br>
바로 안떠올랐던 이유는 1~x의 연속된 수가 있으면 이 수를 다더한 값까지의 연속된 수를 </br>
들수 있다는걸 못떠올려서 였던거 같다. </br>
즉 1,2,3,4 가있으면 1+2+3+4=10이므로 1부터 10까지 모든 수를 만들수 있다. </br>
이를 가지고 코드를 구현했다. </br>
```python
N = int(input())
coins = list(map(int,input().split()))

coins.sort()
current_range = 0

for coin in coins:
  print('coin cu',coin, current_range)

  if coin <= current_range+1:
    print('연속가능')
    current_range += coin
    
  else:
    break

print(current_range+1)
```

- 처음에는 가능한 범위를 범위로 저장하려고 했다. </br>
그래서 배열로 관리하려하니 꼬였다. </br>
간단하게 current_range를 변수로 관리하였다. </br>

- 1-N 까지의 연속된 수를 만들수 있는 상태라면 </br>
다음 수 x가 N+1보다 작거나 같으면 x가 들어와도 연속을 유지한다. </br>
이때 연속 가능 범위는 1-x까지를 다 더한 수가 된다. </br>

- 첫시작 부분 때문에 아이디어 얻는데 어려웠다. </br>
1만 있는경우를 1~1까지 연속된 수라 생각하고 </br>
다음수 2는 1+1인 2보다 작거나 같기때문에 연속을 유지한다. </br>

- 2로 시작하는 경우 논리 </br>
current_range의 의미 정의를  </br>
1부터 current_range까지 연속된 수를 만들수 있다. 로 해야한다. </br>
그렇기에 1이 들어오면 1-1이 되는거고 0인 상태에서는 1-0으로 성립하지않는다. </br>
즉 2로 시작하는 경우 2~0이 되기때문에 성립하지 않는다. </br>
