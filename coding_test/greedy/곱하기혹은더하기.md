## 문제
각 자리가 숫자(0부터9)로만 이루어진 문자열 S  </br>
왼쪽부터 오른쪽으로 하나씩 모든 숫자 확인하며  </br>
숫자사이에 'x'혹은 '+' 연산자를 넣어서 만들 수 있는 가장 큰수를 구하라 </br>
단 모든 연산은 왼쪽부터 순서대로 이루어짐 </br>

## 입력
첫째 줄에 여러 개의 숫자로 구성된 하나의 문자열 S가 주어진다. 

## 출력
첫째 줄에 만들어질 수 있는 가장 큰 수를 출력한다.

## 풀이
떠오른 방법은 </br>
- 0, 1이 포함된 계산은 '+' 나머진 '*'로 계산,
- 하나씩 계산할때마다 '+' '*' 둘다 해보고 큰 수로 결정

더 효율적으로 보이는 첫번째로 시도  </br>

```python
S = list(map(int,input()))

print(S)

result = 0

for s in S:
  print(s)
  if result == 0 or result == 1 or s == 0 or s == 1:
    print(result,'+',s)
    result += s
    

  else:
    print(result,'*',s)
    result *= s
    

  print('re',result)
```

보완 </br>
조건을 부등호로 주면 더 간단히 나타낼 수 있음 </br>
result <= 1 이런식으로 </br>
